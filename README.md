# Elo+Bar Win Prediction Model
This model is a combination of Elo Ratings and BAR (box score above replacement player) that can predict live volleyball games with an accuracy range of 60~70%

# Elo Ratings
Elo ratings were originally created by prof. Arpad Elo. His ratings system was originally created for rating chess players in a mathematically logical method. This chess rating system prevailed around people and was later developed further and tweeked for predicting games in baseball, basketball, football, and etc.
